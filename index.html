Reddit post karma trajectories

Results from five subreddits

I am collecting data on the progress of posts through the following subreddits:
  • The_Donald
  • gonewild
  • AskReddit
  • IAmA
  • dataisbeautiful

To do this, I wrote a python script which reads the top 100 posts in each of these subreddits every two minutes and dumps the results into a sqlite database.

To visualise the results, I then wrote an R script which can select a sample of these posts and display them in three dimensions - for example by post age in days, post score (net "karma" - how many times more it's been upvoted than downvoted) and number of comments. That's all very well, but you can only really get a feeling for the data if you can rotate the 3D plot. Fortunately, R also comes with a library called rglwidget (now part of the RGL library) which can render a 3D plot as a Javascript object.

The 3D objects are a bit heavy in terms of RAM requirements, so I'm not putting them all on the same page.

The graphs

All the subreddits in one graph

This is an overview chart. I've plotted all five of the subreddits on one 3D canvas, showing the relationship between the age of the post in days, the rank or position in its subreddit, and the net karma score. You can see that posts on The_Donald don't hang around long before being pushed out by new posts. GoneWild and AskReddit posts have similar trajectories to each other, lasting about twice as long as T_D posts. IAmA and dataisbeautiful, on the other hand, have much longer-lived posts: there's less of a constant stream of new material, so a popular post can be in the top 100 for about 5 days on dataisbeautiful, or up to 8 days for IAmA.
